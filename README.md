# Balance Game

간단한 밸런스 게임 웹앱입니다. 임의의 질문을 뽑아 A/B 중 하나를 선택하며 즐길 수 있고, 필요에 따라 상황별 질문만 필터링할 수 있습니다.

## 주요 기능
- **랜덤 질문**: 각 카테고리별 질문을 무작위로 노출합니다.
- **상황 필터**: 전체/일반/커플/썸/직장 동료/친구/가족 카테고리를 드롭다운으로 선택할 수 있습니다.
- **공유 기능**: 현재 질문을 Web Share API 혹은 클립보드 복사로 손쉽게 공유합니다.
- **질문 데이터 확장**: 카테고리별로 수십 개의 신선한 질문이 포함되어 있습니다.

## 실행 방법
1. 이 저장소를 클론하거나 다운로드합니다.
2. 정적 파일이므로 별도 빌드 없이 브라우저에서 `index.html`을 열면 됩니다.
3. 로컬 서버가 필요하다면 다음 명령으로 간단히 실행할 수 있습니다.

   ```bash
   npx serve .
   ```
